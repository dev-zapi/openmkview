API 端点:

  • GET /api/projects - 获取项目列表
  • POST /api/projects - 创建项目
  • GET /api/files/tree?project_id=1 - 获取文件树
  • GET /api/files/content?path=README.md&project_id=1 - 获取文件内容
  • GET /api/settings - 获取设置
  • PUT /api/settings - 更新设置
